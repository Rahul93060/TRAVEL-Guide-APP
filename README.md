# 🌍 Travel Guide App

A simple Android application built using **Jetpack Compose** that displays a list of tourist cities with a splash screen.

---

## 📱 Features

* ⏳ Splash Screen with 5-second delay
* 🔀 Navigation using NavController
* 🏠 Home Screen with scrollable list
* 📜 Displays tourist cities using LazyColumn
* 🎨 Built with modern UI (Jetpack Compose)

---

## 🛠️ Tech Stack

* **Kotlin**
* **Jetpack Compose**
* **Navigation Component**
* **Material 3**

---

## 🚀 App Flow

1. App starts with Splash Screen
2. Displays welcome message
3. Waits for 5 seconds
4. Navigates to Home Screen
5. Shows list of tourist cities

---

## 📂 Project Structure

* `MainActivity.kt` → Entry point of app
* `TravelGuideApp()` → Navigation setup
* `SplashScreen()` → Splash logic with delay
* `HomeScreen()` → Displays city list

---

## 📸 Screens

* Splash Screen
* Home Screen (Scrollable List)

---

## ▶️ How to Run

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on Emulator or Device
